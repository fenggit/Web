Mobx 用来替代 redux，是一个非常强大的状态管理工具。

Mobx 学习文档：https://cn.mobx.js.org/

[Mobx官网](https://zh.mobx.js.org/README.html)



**Mobx VS redux**

- Mobx 更偏向面向对象的写法
- Mobx 可以对一份数据直接修改，不用返回一个新的数据
- Mobx 可以多个store
- Mobx 默认以javascript原生方式存储数据，可观察对象



安装 Mobx

```
npm install --save mobx
```
